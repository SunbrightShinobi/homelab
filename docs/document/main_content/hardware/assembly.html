1.3.2. Hardware Assembly¶

1.3.2.1. Dell Precision 3431, Hardware Assembly of gerudo¶

Important

These assembly steps and images are intended to be used by someone with previous experience and knowledge of electronics assembly including following proper ESD procedures and are provided for use at user’s own risk.

See Reference: OEM Dell Precision 3431 Workstation Service Manual

  1. Remove top cover from system by sliding Blue Lever at rear of system to right. Then use grip on cover to slide cover rearward from front of system to where it can be lifted up and set aside.

    Figure 1.1 Dell Precision 3431, Assembled, Rear View¶

    Mark Complete:

    Hostname

    Print and Sign Name

    Date

    gerudo

  2. Remove Factory 2.5/3.5 inch HDD Mount from factory HDD Mount Cage:

    1. Disconnect SATA Power cable from HDD Mount harness

    2. Using care, remove remaining cabling from being attached to HDD Mount in order for it to be removed without obstacles

      Figure 1.2 Dell Precision 3431, Top Cover Removed¶

    3. Using the lever with arrow on it, pull level towards front of system and lift out HDD Mount and set aside

      Figure 1.3 Dell Precision 3431, HDD Mount Removed¶

    Mark Complete:

    Hostname

    Print and Sign Name

    Date

    gerudo

  3. Remove Factory HDD Mount Cage from system:

    1. Using care, remove cabling from being attached to HDD Cage lever in order for it to be removed without obstacles

      Figure 1.4 Dell Precision 3431, HDD Cage Cabling Unrouted¶

    2. Using care, lift up on three tabs of system front bezel cover, so that it is tilted away from system frame and not locking in HDD Cage edge

    3. Using the lever with arrow on it, pull level towards rear of system and lift out HDD Mount and set aside

      Figure 1.5 Dell Precision 3431, HDD Cage Removed¶

    Mark Complete:

    Hostname

    Print and Sign Name

    Date

    gerudo

  4. Replace (or Install) PCIe NVMe M.2 SSD Drive system:

    1. Using care, unscrew Phillips screw at edge of SSD and set aside

      Figure 1.6 Dell Precision 3431, PCIe NVMe M.2 SSD Replacement/Installation (1 of 4)¶

    2. Using care, lift up on screw edge of SSD to 45 degree angle and then pull out very carefully from PCIe NVMe M.2 connector and set aside. Take care of silicon sponge-like heatsink (If factory installed)

      Figure 1.7 Dell Precision 3431, PCIe NVMe M.2 SSD Replacement/Installation (2 of 4)¶

    3. Using care, remove and install in same location silicon sponge-like heatsink from removed SSD (If factory installed) onto new SSD

      Figure 1.8 Dell Precision 3431, PCIe NVMe M.2 SSD Replacement/Installation (3 of 4)¶

    4. Using care, install new SSD in same manner as removal by inserting device into PCIe NVMe M.2 connector at a 45 degree and ensuring it is seated fully, then tilting down to line up screw mount and install screw.

      Figure 1.9 Dell Precision 3431, PCIe NVMe M.2 SSD Replacement/Installation (4 of 4)¶

    5. Verify SSD is mounted correctly and fully in connector before continuing.

    Mark Complete:

    Hostname

    Print and Sign Name

    Date

    gerudo

  5. Reinstallation of HDD Mount Cage into system:

    1. Ensure factory connected Blue SATA cable to SATA0 port and the Black SATA cable to the SATA3 port next to the PCIe NVMe M.2 SSD connector as shown in previous section.

      Figure 1.10 Dell Precision 3431, Verify SATA Connections¶

    2. Reinstall HDD Mount Cage by inserting at a angle to the side of system, ensuring metal tabs slide into position.

    3. Then rotate HDD Mount Cage down until flat in position.

    4. Reinstall system front bezel cover, ensuring all three tabs are secured

    5. Using the lever with arrow on it on HDD Mount Cage, pull level towards front of system to fully secure cage.

      Figure 1.11 Dell Precision 3431, HDD Mount Cage Reinstalled¶

    Mark Complete:

    Hostname

    Print and Sign Name

    Date

    gerudo

  7. Replacement or Installation of NVIDIA Tesla P4 8GB GPU, PCIe 3.0, 16x

    Important

    The NVIDIA Tesla P4 GPU is meant for a system with an active forced air cooling system to push in cool air to inlet of GPU. For a normal enterprise datacenter server this airflow is built into the server but the Dell 3431 Workstation is not an enterprise server. I am testing installation of the Tesla P4 as-is since its installed position is directly in front of the main systme intake fan that hopefully will not require an inlet duct for more direct air flow force.

    1. Replace the full profile bracket with the low profile bracket on the NVIDIA Tesla P4 GPU with the (2) front screws and (2) rear screws on backplate shown in the following figures.

      Figure 1.18 Dell Precision 3431, NVIDIA Tesla P4 GPU Profile Bracket (1/2)¶

      Figure 1.19 Dell Precision 3431, NVIDIA Tesla P4 GPU Profile Bracket (2/2)¶

    2. Remove currently installed GPU (If there is one) then replace with NVIDIA Tesla P4 PCIe 3.0 GPU in the PCIe 3.0, 16x slot.

      Figure 1.20 Dell Precision 3431, NVIDIA Tesla P4 GPU Installed (1/2)¶

      Figure 1.21 Dell Precision 3431, NVIDIA Tesla P4 GPU Installed (2/2)¶

    Mark Complete:

    Hostname

    Print and Sign Name

    Date

    gerudo

  8. Installation of Intel X520-DA2 10Gbe NIC, PCIe 2.0, 8x into PCIe 3.0, 4x slot per the images below

    Important

    The Intel X520-DA2 10Gbe NIC is designed for a PCIe 2.0, 8x but the Dell 3431 PCIe 3.0, 4x slot is open ended so a 8x card will fit. I am still testing if there are bandwidth limitations by installing it in a PCIe 3.0, 4x slot. It is compatible and usable by VMware vSphere ESXi but needs to be load tested to determine if both 10Gbe SFP+ port can support 10Gbe concurrently.

    Figure 1.22 Dell Precision 3431, Intel X520-DA2 10Gbe NIC Installation (1/3)¶

    Figure 1.23 Dell Precision 3431, Intel X520-DA2 10Gbe NIC Installation (2/3)¶

    Figure 1.24 Dell Precision 3431, Intel X520-DA2 10Gbe NIC Installation (3/3)¶
